# Help Desk Ticketing System

This project is a self-hosted help desk system built using Docker and osTicket.

I deployed it on a cloud server and connected it to a MySQL database. Users can submit support tickets through the website, and I respond to them through the admin panel.

## Features
- Users can submit support tickets
- Admin panel to manage and respond to tickets
- Real user interactions and responses

## Technologies Used
- Docker
- MySQL
- Linux

## How to Run

1. Install Docker and Docker Compose  
2. Run:
   docker-compose up -d  
3. Open browser:
   http://localhost:8080  

This system was tested with real users submitting tickets and receiving responses.
